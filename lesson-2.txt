1. Giải thích các câu lệnh: git init, git add <file_name>, git add ., git commit -m “message”.
Trả lời
- git init: Dùng để khởi tạo respository cho thư mục hiện tại
- git add <file_name>: Thêm file hiện tại vào staging area
- git add . : Thêm toàn bộ các file có thay đổi vào staging area
- git commit -m “message”: Tạo một commit với ghi chú 

2. Các file theo từng vùng:
a. Tạo 3 file: file1, file2, file3 - > Cả 3 file đều ở Working Directory
b. Chạy lệnh: git init -> Cả 3 file đều ở Working Directory
c. Chạy lệnh: git commit -m”init project” -> Vì không add file nên cả 3 file vẫn ở Working Directory
d. Chạy lệnh: git add file1 -> File1 chuyển qua Staging area, 2 file còn lại không thay đổi 
e. Chạy lệnh: git commit -m”add file1” -> File1 được chuyển qua Repository 
f. Chạy lệnh: git commit -m”add file”  -> Không có gì thay đổi
